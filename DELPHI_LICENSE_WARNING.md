# Delphi Community Edition 라이센스 주의사항

**작성일**: 2026-01-01  
**중요도**: ⚠️ **매우 중요**

---

## ⚠️ **중요: "무료"의 함정**

### **Delphi Community Edition 라이센스 제한**

Embarcadero는 "무료"라고 광고하지만, **실제로는 많은 제약**이 있습니다.

---

## 📋 **라이센스 제한 사항**

### **1. 수익 제한**
```
❌ 연간 수익 $5,000 이하만 사용 가능
❌ 회사 수익이 아닌 개인/팀 수익 기준
❌ 오픈소스 프로젝트도 수익 발생 시 제한
```

**실제 의미**:
- 개인 취미 프로젝트: ✅ OK
- 소규모 사이드 프로젝트: ⚠️ 조건부
- 상업용 개발: ❌ 불가능
- 회사 업무: ❌ 불가능

### **2. 사용 목적 제한**
```
❌ 상업적 용도 제한
❌ 기업 환경 사용 제한
❌ 프리랜서 업무 제한
```

### **3. 기능 제한**
```
❌ 명령줄 컴파일 불가 (IDE만 가능)
❌ 일부 고급 기능 제한
❌ 플랫폼 제한 (Windows만)
```

### **4. 배포 제한**
```
⚠️ 컴파일된 바이너리에 제한 없음
⚠️ 하지만 개발 과정에서 라이센스 위반 가능
```

---

## 💡 **실제 사용 가능한 대안**

### **Option 1: Free Pascal (FPC) - 진짜 무료 ✅**

#### **라이센스**
- ✅ **완전 무료** (GPL/LGPL)
- ✅ **상업적 사용 가능**
- ✅ **수익 제한 없음**
- ✅ **기업 사용 가능**
- ✅ **오픈소스**

#### **기능**
- ✅ 명령줄 컴파일 가능
- ✅ 크로스 플랫폼 (Windows, Linux, macOS)
- ✅ Delphi 호환 문법
- ⚠️ Inno Setup 전체 빌드는 불가 (VCL 의존성)

#### **사용 가능 범위**
```
✅ 테스트 프로그램 컴파일
✅ 유틸리티 도구 개발
✅ DLL 로딩 검증
✅ 프로토타이핑
❌ Inno Setup 전체 빌드 (VCL 필요)
```

---

### **Option 2: Lazarus IDE - 진짜 무료 ✅**

#### **라이센스**
- ✅ **완전 무료** (GPL/LGPL)
- ✅ **상업적 사용 가능**
- ✅ **수익 제한 없음**
- ✅ **기업 사용 가능**

#### **기능**
- ✅ 비주얼 디자이너 포함
- ✅ LCL (Lazarus Component Library)
- ✅ 크로스 플랫폼
- ✅ Free Pascal 기반

#### **사용 가능 범위**
```
✅ GUI 애플리케이션 개발
✅ 테스트 프로그램 개발
✅ 상업용 프로젝트
⚠️ Inno Setup은 Delphi VCL 의존 (LCL 호환 안 됨)
```

---

### **Option 3: Delphi Professional (유료) 💰**

#### **가격**
- 💰 **약 $2,000+** (연간 라이센스)
- 💰 영구 라이센스는 더 비쌈

#### **장점**
- ✅ 모든 제한 없음
- ✅ 상업적 사용 가능
- ✅ 명령줄 컴파일 가능
- ✅ 공식 지원

#### **단점**
- ❌ 매우 비쌈
- ❌ 개인 프로젝트에는 과함

---

## 🎯 **Inno Setup 개발을 위한 현실적 선택**

### **시나리오 1: 개인 취미 프로젝트**
```
권장: Delphi CE
- 수익 $5,000 이하면 합법
- Inno Setup 전체 빌드 가능
- 무료
```

### **시나리오 2: 오픈소스 기여**
```
권장: Delphi CE (조건부)
- 수익 발생 안 하면 OK
- 기여만 하고 수익 안 내면 합법
- 하지만 애매한 영역
```

### **시나리오 3: 상업적 개발**
```
필수: Delphi Professional
- Community Edition은 라이센스 위반
- 법적 리스크
- 정식 라이센스 구매 필요
```

### **시나리오 4: 테스트만 필요**
```
권장: Free Pascal
- 완전 무료, 제한 없음
- 테스트 프로그램 컴파일 가능
- SimpleDLLTest.pas 실행 가능
```

## ⚠️ **라이센스 위반 리스크**

### **Delphi CE를 상업적으로 사용하면**
```
❌ 라이센스 위반
❌ 법적 조치 가능
❌ 벌금 가능
❌ 소프트웨어 사용 금지
```

### **실제 사례**
Embarcadero는 라이센스 위반을 적극적으로 추적하며, 특히:
- 회사 이메일로 등록
- 회사 IP에서 사용
- 상업용 제품 배포

이런 경우 감사 대상이 될 수 있습니다.

## 💡 **우리 프로젝트의 권장 사항**

### **현재 상황**
- ✅ 코드는 100% 완성
- ✅ 로직은 검증 완료
- ⏸️ 컴파일만 필요

```bash

https://www.freepascal.org/download.html

# 간단한 테스트
cd Projects\Tests
fpc SimpleDLLTest.pas
SimpleDLLTest.exe
```

**장점**:
- ✅ 완전 무료
- ✅ 라이센스 걱정 없음
- ✅ DLL 로딩 검증 가능

#### **2단계: Delphi CE (개인 프로젝트만)**
```
조건:
- 개인 취미 프로젝트
- 연 수익 $5,000 이하
- 비상업적 용도
```

**주의**:
- ⚠️ 회사 업무면 안 됨
- ⚠️ 수익 발생하면 안 됨
- ⚠️ 프리랜서 업무면 안 됨

#### **3단계: 상업적 사용 시**
```
필수: Delphi Professional 구매
```
